## Proyectos

### Sangre y Ceniza — E-commerce con Django

Proyecto final del bootcamp: aplicación e-commerce con autenticación por roles, catálogo de productos persistido en base de datos, carrito de compras y confirmación de órdenes con simulación de pago.

- **Tecnologías:** Python, Django, SQLite, Bootstrap 5
- **Funcionalidades destacadas:** autenticación con roles, CRUD de productos, carrito de compras, flujo de pago simulado
- - **Repositorio:** https://github.com/Xiihib/sangre_y_ceniza
